Git username & git email conf 
```bash
git config --add --global user.name "github_username_here"
git config --add --global user.email "email@example.com"
```

Git github default branc
```bash
git config --global init.defaultBranch main
```


**CLI ile login ol**  

```bash
gh auth login
```
